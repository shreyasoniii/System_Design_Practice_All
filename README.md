# System Design Practice

This repository contains my Low-Level Design (LLD) and System Design practice problems and solutions.

The purpose of this repository is to improve my object-oriented design skills, understand design patterns, and prepare for software engineering interviews.

Topics covered:

* Low-Level Design (LLD)
* Object-Oriented Design (OOD)
* Design Patterns
* Java-based System Design Problems

Some examples include:

* Snake and Ladder
* Parking Lot
* Tic Tac Toe
* Library Management System
* Other commonly asked LLD interview questions

I will continue adding more problems and solutions as I learn and practice.

Happy Coding! 🚀
